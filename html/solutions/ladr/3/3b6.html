Question

Prove there does not exist \(T \in \mathcal{L}(\mathbf{R}^5)\) such that \(\mathcal{R}(T) = \mathcal{N}(T)\).

Solution:

From rank-nullity theorem we know,

\[ \text{dim } \mathbf{R}^5 = \text{dim }\mathcal{N}(T) + \text{dim }\mathcal{R}(T)\]

because \(\mathcal{R}(T) = \mathcal{N}(T)\), then dim of each of \(\mathcal{R}(T)\) and \(\mathcal{N}(T)\) is 2.5.

Which is not possible because dimension is a non-negative integer by definition.

Hence, there does not exist \(T \in \mathcal{L}(\mathbf{R}^5)\) such that \(\mathcal{R}(T) = \mathcal{N}(T)\). [QED]
